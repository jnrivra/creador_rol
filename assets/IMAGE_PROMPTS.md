## Notas de Generación

- **Modelo recomendado:** DALL-E 3, Midjourney v6, o Stable Diffusion XL
- **Negatives:** text, words, letters, humans, realistic, photographic, dark, scary
- **Post-proceso:** Exportar como `.webp` con calidad 80-85 para equilibrar calidad/peso
- **Fondos:** Máx ~200KB cada uno para carga rápida
- **Retratos:** Máx ~80KB cada uno
